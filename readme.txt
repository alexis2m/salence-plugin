This plugin allows you to add the Salence.io Pixel code to your WordPress website

== Description ==

This plugin has beed developed so Salence.io users can add their notification pixel to their website easily and quickly.

= Features of Insert Headers and Footers =

* One click Install
* Friction-Less

== Installation ==

1. Install Salence.io Pixel by uploading the `salence` directory to the `/wp-content/plugins/` directory.
2. Activate Salence.io Pixel through the `Plugins` menu in WordPress.
3. Insert your Salence Pixel by going to the `Settings > Salence.io Pixel` menu.
